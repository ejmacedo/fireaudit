Conecte seu primeiro pfSense

Leva cerca de 5 minutos. O agente só lê configurações, nunca modifica nada no seu firewall.

1

Dê um nome a este firewall

Esse nome aparece em todos os dashboards e relatórios.

2

Instale o pacote RESTAPI no pfSense

System > Package Manager > procure "RESTAPI" e instale. Já tem instalado? Pule para o passo 3.

▶ Ver vídeo (2 min)
3

Gere uma API Key para o usuário admin

System > REST API > Keys > Add. Associe ao usuário admin local. Essa chave fica só no seu pfSense — nunca é enviada para a nuvem.

4

Copie o script já com seu token de ingestão embutido

Services > Cron > Add. Rodar a cada 15 minutos.

#!/bin/sh
curl -s -H "X-API-Key: SUA_API_KEY_LOCAL" http://192.168.1.254:8443/api/v2/firewall/rules \
  | curl -s -X POST -H "Authorization: Bearer fa_ingest_8f2a91cd6e" -d @- https://api.fireaudit.io/v1/ingest
Travou em algum passo? Agende 15 min com o suporte (resposta em até 1 dia útil).

Sua Conta Individual já está vinculada a uma empresa

A Conta Individual (Blackwood Retail Group) foi desenhada para 1 empresa só — isso mantém o plano simples e sem excesso de camadas para quem administra o firewall de um único lugar.

Para gerenciar o firewall de outra empresa com o mesmo login, você precisa migrar para uma Conta Multiempresa.

O que muda ao migrar:

  • • Blackwood Retail Group continua exatamente como está, sem perder histórico.
  • • Você paga o mesmo valor do Pro atual pela 1ª empresa — sem penalidade por migrar.
  • • Empresas adicionais entram com desconto por volume (ver tabela de billing).
