Conecte seu primeiro pfSense

Leva cerca de 5 minutos. Siga os passos abaixo — o agente só lê configurações, nunca modifica nada no seu firewall.

1

Instale o pacote RESTAPI no pfSense

System > Package Manager > procure "RESTAPI" e instale. Já tem instalado? Pule para o passo 2.

▶ Ver vídeo (2 min)
2

Gere uma API Key para o usuário admin

System > REST API > Keys > Add. Associe ao usuário admin local.

3

Cole o token de ingestão e a API key abaixo

Vamos gerar o script pronto pra você colar no Cron do pfSense.

4

Copie o script gerado no Cron do pfSense

Services > Cron > Add. Rodar a cada 15 minutos.

#!/bin/sh
curl -s -H "X-API-Key: SEU_TOKEN_AQUI" http://192.168.1.254:8443/api/v2/firewall/rules \
  | curl -s -X POST -d @- https://api.fireaudit.io/ingest/TOKEN_DA_SUA_CONTA
Travou em algum passo? Agende 15 min com o suporte (resposta em até 1 dia útil, fora do nosso horário comercial local).
